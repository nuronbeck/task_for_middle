# apartments_demo (task for job)

## Install node dependencies
```
npm install
```

## Open the folder
```
cd apartments_demo
```

### Run the app
```
npm run serve
```

### Open the app in browser

[http://localhost:8080/](http://localhost:8080/)


### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
